# Bilgisayar Ağları TCP/UDP Hastane Randevu Sistemi

Bu proje, Bilgisayar Ağları dersi kapsamında Python ile geliştirilmiş TCP ve UDP tabanlı bir istemci-sunucu uygulamasıdır.

## Proje Amacı

Projenin amacı, TCP ve UDP haberleşme protokollerini kullanarak doktor-hasta randevu sistemini simüle etmektir.

## Kullanılan Teknolojiler

- Python
- Socket Programming
- TCP
- UDP
- Threading
- Select

## Dosyalar

- `22100011064_Server.py` : Sunucu tarafı kodları
- `22100011064_Client.py` : İstemci tarafı kodları
